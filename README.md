# tunlr vm

## Requirements

* [VirtualBox][vb]
* [Vagrant][vagrant]
* `vagrant box add lucid64 http://files.vagrantup.com/lucid64.box`

## Setup

1. Clone this files
2. `vagrant up`
3. Setup your primary dns to: `192.168.23.23`


[vb]: https://www.virtualbox.org/ "VirtualBox"
[vagrant]: http://vagrantup.com/ "Vagrant"
